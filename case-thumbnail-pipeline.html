// The Problem

Thumbnails break when you scale brands

Running content across multiple brands means generating thumbnails for YouTube, Instagram Reels, TikTok, feed posts, and carousels . all with different dimensions, styles, and brand guidelines. Doing this manually takes 30+ minutes per thumbnail. Outsourcing costs $15-50 per piece and kills iteration speed.

The harder problem: keeping your face consistent across AI-generated images. Standard image generators produce a different face every time. The skill needed a face-locking system that compounds over time.

// The Solution

One pipeline, all brands, all platforms

A multi-agent AI system built inside Claude Code that generates brand-consistent thumbnails across any platform. Give it a topic (or steal a reference thumbnail), and it produces variations with your face, your style, your brand palette.

🎯

Face Locking

Uses approved outputs as face references instead of raw selfies. Consistency compounds with every approved thumbnail.

📋

JSON Prompts

Structured prompts with face_lock, subject position, background style, and typography fields. No guessing.

📱

Multi-Platform

YouTube 16:9, Reel 9:16, Feed 1:1, Carousel 4:5, Blog ~2:1 . one click generates the right dimensions.

🎨

Brand Aware

Each brand gets its own background styles, color palette, and face references. Automatic style matching.

// Prompt Engine

Structured, not random

Every thumbnail uses a JSON-structured prompt. No vague natural language . every element is controlled: subject position, clothing, background style, text placement, and the critical face_lock directive.

{
  "face_lock": "Copy face from first reference exactly",
  "subject": {
    "position": "right side, 40% width, chest-up",
    "pose": "pointing upward"
  },
  "background": {
    "type": "cinematic blurred home office",
    "lighting": "warm natural with soft bokeh"
  },
  "typography": {
    "line_1": "YOUR HOOK TEXT",
    "highlight": "bold black on yellow"
  }
}
