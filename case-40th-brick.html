// The Concept

A birthday gift designed as an interactive experience

A personalized birthday gift, engineered as a playable puzzle adventure. 40 rooms, one for each year. 4 themed chapters that progress in difficulty and narrative complexity. Solo and multiplayer modes for shared play. Fully responsive across desktop, tablet, and mobile.

The constraint was personal. It had to feel crafted, not generic. Each room carries unique puzzle mechanics. The experience is linear but varied, with no two rooms sharing the same interaction pattern.

// Engineering

Zero frameworks. One session. Pure craftsmanship.

Built entirely with Claude Code in a single engineering session. No frameworks, no libraries. Pure JavaScript with HTML Canvas rendering. Every room has its own puzzle logic, visual style, and interaction model.

The architecture is deliberately minimal. A Node.js and Express server handles multiplayer state. The client is vanilla JavaScript with Canvas API for all rendering. Responsive design ensures consistent gameplay across screen sizes, from desktop monitors to mobile phones.

40 Unique Rooms

Each room features distinct puzzle mechanics. Pattern matching, timing challenges, spatial reasoning, memory sequences, and more.

4 Themed Chapters

Progressive difficulty curve across four narrative arcs. Each chapter introduces new interaction patterns and visual themes.

Zero Dependencies

No React. No Phaser. No game engine. Pure JavaScript and HTML Canvas. The entire game runs without a single external library.

Multiplayer Support

Real-time co-op mode via WebSocket connections. Shared puzzle state, synchronized progress, and collaborative solving.
